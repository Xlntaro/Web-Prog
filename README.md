# Трекер Витрат "Скарбничка" 💰

Стильний та сучасний веб-додаток для обліку особистих фінансів. Проєкт розроблено як курсову роботу / лабораторну з веб-програмування та стандартизації.

## Опис проєкту

Додаток побудовано за архітектурою **Single Page Application (SPA)** з використанням найсучасніших веб-стандартів. Він дозволяє користувачам легко додавати витрати, розбивати їх за категоріями та відслідковувати свій загальний баланс. Усі дані безпечно зберігаються локально в браузері.

### Основні можливості (Features)
- 🚀 **SPA Маршрутизація:** Перемикання між екранами ("Головна", "Транзакції", "Налаштування") без перезавантаження сторінки.
- 🎨 **Сучасний дизайн (Glassmorphism):** Преміальний зовнішній вигляд з ефектом напівпрозорого скла, темною темою та плавними CSS-анімаціями.
- 💾 **Локальне збереження даних:** Використання `localStorage` для збереження історії витрат між сесіями.
- ♻️ **Реактивний стан (State Management):** Власний міні-store (подібний до Redux/Vuex), що автоматично оновлює всі необхідні компоненти на екрані при додаванні або видаленні транзакції.

## Технологічний стек
Проєкт не використовує важких фреймворків чи Node.js-збирачів, демонструючи глибоке розуміння нативних веб-технологій:
- **HTML5** (Семантична розмітка)
- **CSS3** (CSS Variables, Flexbox, Grid, Animations, Backdrop Filter)
- **Vanilla JavaScript (ES6+)**
- **Native ES Modules** (Нативні імпорти/експорти `import/export` прямо в браузері)

## Як запустити локально

Оскільки проєкт використовує Native ES Modules, він повинен запускатися через локальний веб-сервер (пряме відкриття `index.html` через `file://` заблокує імпорти через політику CORS браузера).

1. Склонуйте репозиторій або завантажте файли.
2. Відкрийте термінал у папці з проєктом.
3. Запустіть будь-який простий локальний сервер. Наприклад, за допомогою Python:
   ```bash
   python3 -m http.server 8080
   ```
   Або за допомогою Node.js (якщо встановлено `serve`):
   ```bash
   npx serve .
   ```
4. Відкрийте у браузері посилання: `http://localhost:8080`
